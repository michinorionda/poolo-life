# POOLO 旅LP デザイン仕様書

鞆の浦の申込サイト（[index.html](index.html)）の設計図です。
このMarkdown1枚を渡せば、同じデザインで別の旅LPを作れます。
「次の旅でも使い回す共通テンプレート」として持っておくのが目的です。

> **使い方**：新しい旅を作るときは、この仕様書をベースに
> 「§6 差し替えチェックリスト」の項目だけ入れ替えればOKです。

---

## 1. このLPのコンセプト

- 雑誌の特集扉のような、**静かで余白の多い1ページ完結**のランディングページ
- 縦スクロールで「世界観 → 体験 → 写真 → スケジュール → ガイド → 概要 → 申込」へ自然に誘導
- 申込ボタンはGoogleフォームに直結。スマホでは画面下に**追従する申込バー**が出る
- POOLO / TABIPPO のブランドに準拠（瀬戸内ネイビー × POOLOブルー）

---

## 2. デザインシステム（色・文字・装飾）

### カラー
| 役割 | 色コード | 用途 |
|---|---|---|
| メインブルー | `#189BBC` | POOLOメインカラー。ボタン・帯・アクセント |
| ディープブルー | `#0F7E9B` | ボタンhover、強調文字 |
| インク（濃紺） | `#15323D` | 基本の文字色、暗い背景セクション |
| インクソフト | `#3A5562` | 補助テキスト |
| ペーパー（紙白） | `#F6F4EE` | 背景のベース色 |
| ライン | `rgba(21,50,61,.14)` | 区切り線・枠線 |

### フォント（Google Fonts）
- **見出し（和文）**：Noto Serif JP（明朝）… 上品さ・特集感を出す主役
- **本文（和文）**：Zen Kaku Gothic New / Noto Sans JP（ゴシック）
- **欧文・ラベル**：Jost（細め・字間広め）… "CONCEPT" などの小見出しに
- ※ブランドガイドの和文 ZEN KAKU GOTHIC NEW / 欧文 JOST に準拠

### 装飾のルール
- **角丸**：写真は大きめの角丸（14〜26px）、しずく型・有機的な変形角丸も使用
- **影**：濃いめでぼかし大きく、下方向に落とす（`0 20px 50px -30px rgba(21,50,61,.5)` 系）
- **水彩のにじみ**：背景にブルーのぼかし円をうっすら配置（`watermark`）
- **リビール**：スクロールで各要素がふわっと下から出現（`.rv` クラス）

---

## 3. レイアウト・レスポンシブのルール

- コンテンツ最大幅：**1180px**、左右の余白28px
- **PC ↔ スマホの切替は画面幅 600px / 880px が境界**
- 改行の出し分け（重要）：
  - `<br class="pc-br">` … PCだけ改行、スマホでは詰める
  - `<br class="sp-br">` … スマホだけ改行、PCでは無効
  - `<span class="para-break">` … PCは改行、スマホは1行空け（段落の区切り）
- スマホでは見出しを少し縮小して1行に収める（例：コンセプト見出し）

## 5. 各セクションの「型」（鞆の浦の中身を例に）

### 01 ヒーロー
- 縦長ポスター画像（A4比率）を中央に1枚。角丸＋大きな影。
- 例：「散歩」と「対話」と「問い」と。広島・鞆の浦で過ごす1泊2日。

### 02 コンセプト（中央寄せ）
- ラベル：`CONCEPT`
- アイキャッチ（明朝・ブルー）：*暮らし観光を通じて、自分自身の感性を探究する旅*
- 大見出し（明朝・2行）：*「なにを感じたか」を、ゆっくり言葉にしていく。*
- リード文：2〜4文。スマホでは文ごとに1行空け。

### 03 この旅で体験すること（4カード）
各カード = 縦長写真 ＋ 通し番号(◯) ＋ 短いタイトル ＋ 1文説明
1. 問いを持って、まちを歩く
2. まちの暮らしに触れる
3. 偶然の出会いを楽しむ
4. 感じたことを言葉にする

### 04 ギャラリー
- 12カラムgridに大小8枚を変則配置（`g-a`〜`g-h`）
- スマホでは2枚並び。ラベル：`Scenes of ◯◯`

### 05 スケジュール
- `DAY 1` / `DAY 2` を青いピル（丸タグ）で表示
- 各ステップ = 縦長写真 ＋ 左上に番号バッジ ＋ 工程名（＋時刻）
- 集合・解散の時刻は小さい青字で添える

### 06 ◯◯とは（ストーリー）
- 左：有機シェイプ写真2枚を重ねる／右：テキスト
- 青タグ（テーマ名）＋ 大きな引用（明朝、一部ブルー強調）＋ 補足説明

### 07 ガイド（濃紺背景・反転色）
- 左：顔写真をしずく型に切り抜き、左下に白い名刺風バッジ（名前＋肩書）
- 右：見出し＋プロフィール文（白系文字）
- ※スマホでは縦積み

### 08 宿泊
- 横長カード（左：写真／右：宿名・英字ラベル・説明・料金タグ・公式リンク）

### 09 概要（表）
- `dt / dd` の2列。罫線は破線。
- 行例：企画名・開催日・場所・集合解散・定員・ガイド・参加費・宿泊・主催・協力
- 参加費は数字を特大（Jost）で見せる

### 10 CTA / フッター
- 締めの一文（明朝、キーワードをブルー）＋ 申込ボタン
- フッター：地名ワードマーク（Jost・字間広め）＋ `by POOLO` ＋ 日付

## 6. 差し替えチェックリスト（新しい旅を作るとき）

新規の旅LPは、デザインはそのままに以下だけ入れ替えます。

- [ ] **キービジュアル画像**（ヒーロー用ポスター1枚／縦長）
- [ ] **キャッチコピー**（アイキャッチ・大見出し・リード文）
- [ ] **体験4項目**（タイトル＋説明＋写真4枚）
- [ ] **ギャラリー写真8枚**
- [ ] **スケジュール**（DAY1/DAY2の工程・時刻・写真）
- [ ] **テーマ解説**（◯◯とは）
- [ ] **ガイド情報**（顔写真・名前・肩書・プロフィール）
- [ ] **宿情報**（写真・宿名・料金・リンク）
- [ ] **概要データ**（開催日・場所・定員・参加費・主催・協力 など）
- [ ] **GoogleフォームのURL**（HTML末尾 `const FORM_URL = "..."`）
- [ ] **OGP/ページタイトル**（`<title>` と `og:` メタ）

### 画像の準備
- 写真は横1700px程度にリサイズ、JPEG品質84前後（容量を軽く）
- ヒーローのキービジュアルは別途デザイン（ポスター）を用意
- ガイドの顔写真は明るく、海・まちなど現地感のある背景が◎
