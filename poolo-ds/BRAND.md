# POOLO ブランド & ライティング指針

`poolo-ds` デザインシステムの「思想」の部分。`tokens.css` / `components.css` が**見た目の単一ソース**なら、この BRAND.md は**言葉と間（ま）の単一ソース**。新しい告知ページ・サムネ・OG画像を作るとき、色やコンポーネントと同じ強さでここに従う。

by TABIPPO / POOLO。

## 2. 声のトーン（Voice）

POOLO の言葉は **「煽らない・断定しない・急かさない」**。読み手を追い込まず、内側に問いを置いて帰ってもらう。

### やること
- **短い言い切りを句点で細かく区切り、リズムを作る。** 体言止めと「。」を混在させる。
  > 仕事は頑張ってきた。評価もそれなりにされていて、できることも増えてきている。
- **断定せず、問いに開く。** 口語の自問をそのまま置く。強調は `.quoted`（アクセント色）で。
  > 「これで合っているんだっけ」／「〜って何だっけ…？」
- **転換は一言で、間を作る。** 「ただ、しかし。」のような一言を `.intro-break`（中央・アクセント色・大きめ）で独立させる。
- **CTA も誘い。** 命令形にしない。「問い直してみませんか。」のトーン。
- **内省系の語彙。** 「納得感」「内側の手触り」「言語化」「まだ出会ったことのない自分」。

### やらないこと
- 煽り・恐怖訴求・過度な数字訴求（「今すぐ！」「残りわずか！！」）。
- 事実（先着◯名・参加無料）は**静かに**提示する。強調しすぎない。
- 装飾過多。感嘆符の連打。

## 3. 間（ま）とレイアウトの思想

言葉と同じくらい「余白」が POOLO らしさを作る。

- **ゆったりを終始キープ**：セクション上下 `--space-section-y (72px)`、行間 `--lh-normal (1.9)`〜`--lh-loose (2.15)`、字間 `--ls-base (0.04em)` 基準。
- **装飾は引き算**：縦書きサイドマーク（`.page-side-mark`）、中黒（`·` = `.section-mark .em`）など、細く静かなあしらいだけ。
- **セクション番号は CSS が描く**：見出し本文に数字を書かない。`.section-title` に `data-num="01"` を渡すと `::before` が連番あしらい（イタリック＋44px下線）を描画する。
- **英字あしらい**：eyebrow・ラベルは英字イタリック＋大文字＋広い字間（`--ls-eyebrow` 0.3em）で欧文アクセント。

## 4. ページ構成の型（告知ページ）

基準ページ（`event-30s-v01-*`）が確立した標準の並び。新規ページはこの型を出発点に、不要セクションを引き算する：

1. Hero D（編集型サムネ `.hd`）
2. イベント要点（`.event-summary`）
3. お知らせ帯（`.notice`）
4. 導入 `01`（`.intro-text` + `.intro-break`）
5. テーマ `02`
6. 当日の流れ `03`（`.flow-list`）
7. 当日の問い（`.worksheet-questions`）
8. 対話のルール `04`（DO/DON'T `.dialogue-rules-grid`）
9. 確認事項 `05`（`.terms-list`）
10. 中間CTA（`.cta-mid`）
11. こんな方へ `06`（`.target-list`）
12. 企画理由 `07`
13. 開催概要（`.details-table`）
14. 最終CTA（`.cta`）＋ 追従CTA（`.float-apply`）
15. フッター（`.footer`、2社ロゴ）

各コンポーネントの最小マークアップは `poolo-ds/README.md` とカタログ `poolo-ds/index.html` を参照。

## 5. 運用の約束

- **トークンとコンポーネントはコピペしない。** 新規ページは `poolo-ds/tokens.css` と `poolo-ds/components.css` を `<link>` で読む。
- **新しい汎用パーツを作ったら DS に還元する。** ページに書きっぱなしにせず `components.css` / `tokens.css` / カタログ（`@dsCard`）/ README に反映し、`/design-sync` で claude.ai/design の Design System へ同期する。
- **このトーンから外れる案は「素材」扱い。** ゼロイチの実験は歓迎だが、標準の告知ページはこの BRAND.md に収束させる。
