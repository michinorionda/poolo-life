POOLO VALUES ／ FOUNDATION

POOLOの土台は、
どこにある？

3層のうち、いちばん下を見る
見えないけれど、
共にあるの根が深いほど、幹は太く、花は大きく咲く。
ANSWER
Being（共にある）が、いちばんの土台。
スタンスもDoingも、共にあるという根がなければ立ち上がれない。
Doingが止まっても、幹と根は残る。幹がぶれても、根があれば立て直せる。
でも、根が枯れたら、何を積んでも倒れる。
根
LAYER 1 ／ BEING
共にある。
何かを一緒にやっていなくても、ひとりでいるときも、卒業後も。在り方としての土台。見えないけれど、全部を支えている。
幹
LAYER 2 ／ STANCE
旅のスタンス（3つ）
自己決定する／自分ごと化する／面白がる。根から立ち上がり、幹として全期間ぶれずに立ち続ける。
枝葉
LAYER 3 ／ DOING
共に旅する → 問う → 創る
幹の上で、時間軸に沿って枝葉が伸び、花が咲く。8ヶ月の深化3段階。
なぜこれが実践上も効くか
ヒアリングで最初に確かめるのは、共にある手応え。
カリキュラムで最初に守るのは、ここにいていい の設計。
不調なメンバーへの介入は、スタンスやDoingで煽らず、まずBeingへ戻す。

根が弱れば、何を積んでも倒れるから。
