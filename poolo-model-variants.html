MAIN MAP — 5 VARIANTS

MAIN MAPのパターン比較
5つの視覚メタファー

同じ構成要素を、違う比喩で描く

← 現行のMAIN MAPへ

どのパターンも、同じ構成要素（探究テーマ・実践3つ・仲間の引力場・複数の到達点・8ヶ月の時間軸）を描いています。 違うのは、何を主役に見せるかとどんな比喩で運動を表すか。 現行の「螺旋型」と比べながら、どれが社長の伝えたい感覚に近いかを選んでください。

PATTERN A 放射分岐型 RADIAL BLOOM

探究テーマを中心に置き、花びら状に複数のルートが放射する。各ルートの途中に「旅・問い・作る」のマーカー、先端に到達点。背景には仲間の引力場が広がる。
→ 一つの種から多方向に芽吹く印象。到達点が「一つではない」ことが最も明確に伝わる。

強み「到達点は一つではない」が最も直感的。対称性があり美しい。
弱み時間軸（8ヶ月の流れ）が描きにくい。中心→外のみで、戻り・反復が表現しづらい。
PATTERN B 編み込み螺旋型 WOVEN SPIRAL

3人の螺旋が互いに編み合う。各自の軌跡が独立しているように見えて、交差点で引力が働き、その後の曲がり方が変わる。実践マーカーは中央人物の螺旋上に。
→ 仲間の引力場がどう軌道を曲げるかが最もはっきり出る。POOLO独自性の核心を描く。

強み「仲間の引力で軌道が曲がる」がこの型で最もよく見える。POOLO独自性の核を描ける。
弱み線が多く複雑。初見で読みにくい可能性あり。実践3つが主役1人の軌跡に限定される。
PATTERN C 川デルタ型 RIVER DELTA

左端の源流（探究テーマ）から右端のデルタ（複数の到達点）へ、川が蛇行しながら進む。途中で支流が分岐し、合流する。実践マーカーは本流の蛇行点に。
→ 運ばれていく感覚が最も強い。時間軸が自然に横方向に流れる。

強み時間軸が一目でわかる。「運ばれていく」感覚が自然。初見で読みやすい。
弱み「仲間の引力で軌道が曲がる」の能動性が弱く、受け身な印象に寄る。
PATTERN D 星座マップ型 CONSTELLATION

濃紺の夜空に星として探究テーマ・実践・仲間・到達点を配置し、線で結んで星座を描く。複数の星座が一つの夜空に並ぶ。
→ 自分だけの星座が浮かび上がる物語感。卒業後も夜空に残る、という余韻が作れる。

強み物語性・余韻が最も強い。卒業後も「夜空は残る」の比喩が使える。ビジュアルが他と全く違う。
弱み反復（螺旋）や時間の流れが静的で伝わりにくい。夜空の背景はブランドと浮く懸念。
PATTERN E 年輪深化型 ANNUAL RINGS

中心に探究テーマを置き、同心円の年輪で8ヶ月を刻む。ルートは内→外へ螺旋で貫く。月ごとに輪が一つ増え、外に向かうほど深まり・広がる。
→ 深まりながら広がるが主役。木の年輪のメタファーで、内発的な成長感が出る。

強み「深まり」と「広がり」が同時に描ける。時間軸が同心円で直感的。内発的成長の比喩が強い。
弱み到達点が外周に並び、「一つではない」が対称的になりすぎる。分岐の強調が弱い。
