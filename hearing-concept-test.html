SCOPEこのヒアリングで確かめたいこと

前回の「体験ヒアリング」と違い、今回はPOOLOの世界観・思想・カリキュラムの中身（共有ハブの素材群）を実際に相手に見せ、言葉と図が旅好き×未認知層にどう届くかを直接確かめます。同時に、未来デザイン更新のための肌の反応と違和感ワードを拾います。

検証したい4つの仮説

仮説4-a / 核の1枚ESSENCE を見せたとき、「自分の話に見える」か「賢そうだけど自分ごとにならない」か
仮説4-b / 受容の言葉VALUES の「そのままで、いい」が、キャリア系の言葉より強く刺さるか
仮説4-c / 変化イメージRIPPLE の3ステージで、8ヶ月後の自分を具体的に想像できるか
仮説4-d / 8ヶ月の重さCURRICULUM の長さ・粒度で、どこに離脱ポイントを感じるか

提示戦略の核

いきなり全8素材は見せません。旅の話で文脈を作ったあと、ESSENCE → VALUES → RIPPLE → CURRICULUM の順で降ろします。抽象的な核の1枚から、受容の言葉、変化イメージ、実際のコミット粒度へ。反応タイプで分岐を設け、理論版や未来デザインv5は「知的関心が強い層への分岐オプション」として扱います。
