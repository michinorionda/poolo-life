POOLOのバリュー

旅の学校の、在り方
3 LAYERED STRUCTURE
この構造の前提

POOLOは旅の学校である。だから、旅のスタンス・マインドがすべての土台になる。
バリューは3層で積み重なる。上の層を、下の層が支える構造。

CORE STRUCTURE

3層の積層

LAYER 1 ／ BEING
存在の土台
そのままで、いい。
未完成でも、不完全でも、ここにいていい。
その受容が、すべての土台になる。
↓ すべての前提
LAYER 2 ／ STANCE
旅のスタンス
8ヶ月のあいだ、ずっと床のように支え続ける3つの態度。旅だけでなく、問いも、共創も、すべてを貫く。
自己決定する
受容があるから、自分で決められる。そのままでいい土台があるから、怖がらずに選べる。
自分ごと化する
誰かの問題ではなく、自分の問題として引き受ける。そこから初めて、問いが湧く。
面白がる
想定外も、予想外も、ぜんぶ面白がる。コントロールを手放す態度。
── この3つが、全期間を床のように支える ──
↑ この床の上を、時系列で歩む
LAYER 3 ／ DOING
体験の深化
スタンスという床の上を、時系列で3歩進む。スタンスが支え続けるから、歩むほどに深まっていく。

IMPORTANT

重要な構造的意味

誤読しないために
スタンスとDoingは「対」ではない。
旅のスタンス3つは、全8ヶ月を床のように支える土台。
Doing 3段階は、その床の上を時系列で深化する3歩。
スタンスは消えない。Doingが進んでも、床はずっとそこにある。

たとえば「序盤＝自己決定／中盤＝自分ごと化／終盤＝面白がる」のようなフェーズ対応ではない。

旅の序盤でも、共創の終盤でも、3つのスタンスはすべて発動している。それが旅の学校の流儀。

APPLICATION

カリキュラムへの接続

育成軸としてのスタンス
3つのスタンスは、全8ヶ月を通じて育て続ける。
各ワークは「どのスタンスを刺激するか」で設計される。
Doingのフェーズは、そのスタンスが発露する場の時系列にすぎない。

これにより、カリキュラムの各ワークは「何をやるか」ではなく、「何を育てるか」で設計できる。
