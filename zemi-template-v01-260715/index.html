01
Introduction

旅先で出会った一皿を、
自分の暮らしに持ち帰る。

タイの市場の、裏路地。常連にまじって食べた、名前も知らない家庭料理があった。

あの味が、忘れられない。

「これ、家で再現できないのかな」。旅から戻るたび、そう思っていました。名所のレストランではなく、誰かの台所にこそ、その土地の暮らしが宿っている気がして。

このゼミは、あなたが旅先で出会った「持ち帰りたい一皿」を起点に、その味を暮らしの中で再現できるところまでいっしょに辿る全4回です。レシピを教わる料理教室ではありません。旅の記憶を手がかりに、自分の台所と土地の食卓をつなぎ直す時間です。

02
Theme

レシピではなく、記憶からはじめる。

上手につくることより、その味とつながり直すこと。

調味料の分量や手順は、調べればいくらでも出てきます。このゼミで扱うのは、その手前。あなたがその一皿と出会った情景を思い出し、味の記憶をたどり、いまの暮らしの道具で再現できる形に落とし込んでいきます。

03
Program

全4回の、流れ。

思い出す → たどる → 再現する → 持ち帰る。旅の一皿を、暮らしの味に。

  • 第 1 回
    旅の一皿を思い出す

    「持ち帰りたい一皿」を、出会った情景ごと言葉にする。参加者どうしで旅の食卓を語り合います。

  • 第 2 回
    その土地の"だし"をたどる

    味の土台を分解。何がその土地らしさを作っているのかを、対話しながら探ります。

  • 第 3 回
    家の道具で再現する

    特別な器具なしで、いまの台所で作れる形に。各自が試作し、持ち寄ります。

  • 第 4 回
    持ち帰りレシピをつくる

    自分の言葉でレシピと物語を1枚にまとめ、暮らしに根づく形にして終わります。
