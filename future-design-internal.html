CORE BRAND

コアブランド定義

旅と、問いと、共創で、想像もしなかった世界と自分に、出会いにいく。
  • POOLOは「自分探し」ではなく「自分試し」
  • 答えは自分の中ではなく、自分と相手の「間」にある

構成：世界観 → 誰に → 設計 → 到達点 → その先 の5部構成

PART 1 — WORLDVIEW

第1部：世界観（何を信じているか）

§01 ビジョン — 2層構造

Big Vision（掲げる旗）

旅で人生を豊かにする人を増やし、未来を共に創る。POOLOを通った一人ひとりが「影響を与える人材」になる。重要なのは影響の循環が続くこと。

Inspire Vision（動かすエネルギー）

世の中を面白がりながら、地に足をつけて、誰かの役に立っている人を増やしたい。

原則：Inspire が先、Big が後。逆にするとエネルギーが枯れる。ビジョンは掲げるものではなく、振り返ったときに見えるもの。

§02 バリュー — Being × Doing

Being（根底の態度）= 共にある：何かを一緒にやっていなくても、卒業後も、すべての土台。

#名称重心内容
01共に旅する序盤身体で体験し、仲間と世界へ飛び込む
02共に問う中盤対話と内省、答えの見えない問いを深める
03共に創る終盤知らない誰かに自分の価値を差し出す（WS・イベント）

※「創る」は序盤から中心行為。重心が移るだけ。

§03 豊かさの定義 — F = G × H

  • G（内的充足）= 自分が好き・得意と感じることがある
  • H（つながり）= それが誰かに届き、反応が返ってくる
  • Gだけ → 自分の世界に閉じる／Hだけ → 自己犠牲／G×H → 持続的豊かさ
Fは固定された到達点ではない。新しい世界＝G更新、新しい人＝H更新。「持続的」とはFを更新し続ける力が内在化すること。

§04 四象限 — 豊かさの地図

外的期待すり合わせ低外的期待すり合わせ高
内的充足 高自分の世界（旅好きの多く／ターゲット）F = G × H ← POOLOの目的地
内的充足 低停滞自己犠牲（外を先にすると落ちる）
原則：内が先、外が後。
PART 2 — FOR WHOM

第2部：誰のためか

§05 ターゲットの4状態

エネルギーはある。でも、行き場がない。
状態声本質
出し切れていない自分はこんなもんじゃない。全部を出し切って、何者かになりたい。未消化のエネルギー
こなしてしまっているいつの間にか全部こなせるようになっていた。でも、それが怖い。消化しすぎて感覚が鈍った
追い込まれている働くに、生きるが追いやられている。わたしは、どう生きたいのか？余白を失い立ち止まれない
成長を追い求めてきた社会的正解を選び続けてきた。なのに、なぜ満ちないのか？達成した先に答えがなかった
4状態は年代で分かれない。25歳でも「成長追求型」、35歳でも「出し切れていない」はある。状態は動的。POOLOはこの4つすべてに応える。
PART 3 — HOW

第3部：どうやって実現するか（POOLOモデル）

§06 旅が開き、POOLOが変える

旅の役割 ＝ フィルターを外す

「はたらく自分」のフィルターが外れ、心が動くことに気づく → Gの種が見える。ただし旅だけでは「楽しかった」で終わる。

POOLOの役割 ＝ 引き出し、差し出す実験

4動作：さらに引き出す → 言語化する → 差し出す → 反応を受け取る。担当：WHAT × WHO／自然発生：WHY／担当外：HOW。

§07 関心探究型 vs 目標逆算型

軸目標逆算型関心探究型（POOLO）
起点3年後にどうなりたい？今、何に心が動く？
プロセスゴール → 逆算 → 計画試す → 反応 → また試す
到達先想定内の自分想像もしなかった自分
POOLOは目標を設定しない。「ありたい姿」は中盤以降に自然発生するもの。

§08 変容のエンジン — 身体と知的冒険の往復

  • 体験する（身体）：旅・越境インタビュー・差し出す実験
  • 言語化する（知的冒険）：1on1・振り返り。「学習」ではなく手探りの「冒険」
  • 差し出す（身体）：仮説を知らない人にぶつける → 反応が返る → 1へ戻る
8ヶ月で何度も回す螺旋。3周目に見える自分は、1周目の自分が想像すらできない自分。原則：身体が先、言葉が後。

§09 5つの役割 — エンジンを回す装置

役割エンジン位置具体
鏡反射（身体）あなたはこう見える（仲間・運営のFB）
反応反射（身体）生の声が返る（ミニ実験・WS）
問い言語化（知的冒険）何が起きた？（1on1・振り返り）
出会い接続（身体）予測不能な他者（越境インタビュー）
伴走接続（身体）8ヶ月の継続（同期30名・1on1）
意図的に不在 = 「導き」（コーチング的な認知介入）。「導き」がないから、体験が主語になる。認知を変えてあげるのではなく、変わる条件を整える。

§10 「想像もしなかった世界と自分」が生まれる2つの場

POOLO内（50人）

想像もしなかった「自分」に出会う。IT・看護・営業・教育・デザイン・公務員…多様な50人が鏡になる。

POOLOの外（越境）

想像もしなかった「世界」に出会う。越境インタビュー・差し出す実験・旅。

構造：内で見つけて、外で試す。

§11 8ヶ月の道のり — 4ステップと螺旋

1
見つける（1〜3ヶ月）

旅の散布・越境インタビュー・「ついつい」から小さなラベル・3ヶ月目で初回の実験

2
名乗る（3〜4ヶ月）

「こう差し出したい」と仮決め。合宿（4ヶ月目頭）で対面。完璧じゃなくていい

3
小さく届ける（4〜6ヶ月）

仲間に差し出すミニ実験。安全な場で失敗する。形を変えて繰り返す

4
知らない人に届ける（7〜8ヶ月）

WS・イベントで社会と接続。反応 → WHAT確定。G→WHAT + H創出

4ステップはリニアに見えるが、中身は螺旋。各ステップの中で反射→言語化→接続の小さな螺旋が回る。

§12 差し出しの2軸分解 — 形 × 深さ

形 3種：コンテンツ型／体験型／対話型

Lv対象具体
Lv.1自分に向けて日記・一人時間の習慣・自己対話
Lv.2仲間に向けてnoteで見せる・一緒に作る・聞いてもらう

PART 4 — DESTINATION

第4部：どうなるか（到達点）

§13 After — 全員に共通する到達点

受け取る人から、創る人になっていく。
  • 8ヶ月前：「面白いのはどこかにある」と思って探していた
  • 今：「面白いのは自分で創っていく」仲間と一緒に、知らない誰かに届ける
  • 予想しなかった反応 → 予想しなかった自分が動き始める
  • 消費する側から、創造する側へ。立ち位置が変わる。POOLO終了後も不可逆

§14 影響の循環 — AfterがVisionを実現する

After到達者 → 次の誰かに影響 → その人がまた誰かを動かす

  • イベントを開く人 → 来場者200人が影響
  • ZINEを作る人 → 読者が自分も書き始める
  • スナックを開く人 → 常連が自分の場を作る
  • 地域案件を始める人 → 参加者が地域のファンに
一人ひとりが完結しない。影響が循環することが「未来を共に創る」の正体。
PART 5 — BEYOND

第5部：この先の景色

§15 社会運動としてのPOOLO — 旅を消費から文化に戻す

問題意識：旅は「消費」になってしまった（手軽な代替地・観光地・SNS映え・リゾートバイト）。誰かが作った体験を受け取る → 楽しい → 日常に戻る → 何も残らない → 次の旅を予約。

本来の旅 ＝ 文化：知らない場所に身体で飛び込む → 人と会い → 何かを持ち帰る → 言葉にする → 誰かに差し出す → 自分の中で発酵 → また次の旅へ。旅は受け取るのではなく、自分から何かを差し出す行為だった。
POOLOがやろうとしていること ＝ 旅を消費から文化に戻す

POOLOが育てる文化：個性が集まり、関係の中で価値が巡る。

  • 「みんなちがってみんないい」ではなく、一人ひとりが自分の個性を差し出し、結果が偶発的に立ち上がる運動体
  • 誰かひとりが設計したわけではなく、POOLOそのものという文化が根付く
  • 文化が根付く場では、関係性の中で価値が巡る（お金だけでなく、スキル・信頼・機会・出会い）
  • これが経済の原点 → 文化51% 経済49%（群言堂の考え方）
この場に必要なのは「差し出す覚悟」。ゆるい場でもなく、厳しい場でもない。全力を出しても安全な場。

§16 想像もしなかった世界に出会いにいく

第1層：旅と自分の延長線上

旅そのもののひらき（ふるさと・観光・地域）／生き方（人生・キャリア・仕事）／つながり（コミュニティ・幸せ・グローバル）

第2層：想像もしなかった世界への越境

社会の仕組み（教育・探究）／経済と表現（マーケ・お金・エンタメ）／つくり・表し（アート・写真・ものづくり）／暮らし（食・農・環境）／社会に届ける（ソーシャル・まちづくり）

一人では絶対に出会わなかったテーマに、50人の仲間を通じて出会う。旅がレンズになり、どんなテーマも「自分ごと」になる。
CONCLUSION

結語：POOLOの未来デザインが目指すもの

旅と、問いと、共創で、想像もしなかった世界と自分に、出会いにいく。
  • 共にあるの態度の上に、共に旅し、共に問い、共に創る
  • 一人ひとりが自分の個性を差し出し、その差し出しが関係の中で巡る文化になっていく
  • 旅を消費から文化に戻し、世の中を面白がる人を増やす
  • 飛び込んで、面白がり、共に創るような人を世の中に増やしていく
→ これが「あたらしい旅の学校」の意味の核
