# 東川の旅｜事前交流会 出欠アンケート（Googleフォーム設問文）

> Googleフォームにそのまま転記して使える設問文一式です。
> 事前交流会は **オンライン開催想定**。LINEグループURL・Zoom URL・SlackリンクなどはフォームURL確定後に差し込んでください（本文中は【　】で記載）。

---

## フォーム設定メモ（転記前に）

- **回答の収集**：1人1回答（必要に応じて「回答を1回に制限」をON ／ メールアドレス収集ONでもOK）
- **想定回答者**：東川の旅 参加確定メンバー
- **回答期限の目安**：7/14開催に間に合うよう、**7/10（金）まで**を推奨

---

## フォームタイトル

```
【東川の旅】事前交流会 出欠アンケート
```

## フォーム説明文

```
東川の旅にお申し込みいただき、ありがとうございます！

本番（8/21〜23）の前に、参加メンバーで顔合わせをする「事前交流会」をオンラインで開催します。
当日いきなり会うよりも、一度ゆるっと話しておくと、3日間がぐっと過ごしやすくなります。

開催候補は 7/14（月）または 7/16（水）の夜。
みなさんの都合を教えていただき、参加者が多い日に決定します。

所要時間は1分ほど。お気軽にご回答ください。

──────────
【連絡ツールについて】
事前交流会のあと、参加者専用の Slack を立ち上げてご案内していきます。
Slackの準備が整うまでは LINEグループ でやりとりします。
まだLINEグループに入っていない方は、フォーム最後のリンクからご参加ください。
──────────
```

## 設問1（必須）｜お名前

- **質問**：お名前を教えてください
- **タイプ**：記述式（短文）
- **必須**：はい

```
お名前を教えてください
※フルネームでお願いします
```

---

## 設問2（任意）｜呼ばれたいニックネーム

- **質問**：当日、呼ばれたいニックネーム（あれば）
- **タイプ**：記述式（短文）
- **必須**：いいえ

```
当日、呼ばれたいニックネームがあれば教えてください
（例：たびっぴー、みっちー など）
```

---

## 設問3（必須）｜事前交流会の出欠・参加できる日

- **質問**：事前交流会に参加できる日を教えてください（オンライン開催・夜を予定）
- **タイプ**：ラジオボタン（単一選択）
- **必須**：はい

```
事前交流会に参加できる日を教えてください
※オンライン開催／夜の時間帯を予定しています

◯ 7/14（月）の夜なら参加できる
◯ 7/16（水）の夜なら参加できる
◯ どちらの日でも参加できる
◯ どちらの日も都合がつかない（後日アーカイブ等で共有希望）
```

> 補足：「両日OK」の票が分散しないよう、**単一選択**で作るのがおすすめです。
> 複数日に丸をつけてほしい場合は「チェックボックス（複数選択可）」に変更してください。

---

## 設問4（任意）｜参加しやすい開始時間

- **質問**：参加しやすい開始時間があれば教えてください
- **タイプ**：ラジオボタン（単一選択）
- **必須**：いいえ

```
参加しやすい開始時間があれば教えてください（夜の時間帯で調整します）

◯ 20:00ごろ
◯ 21:00ごろ
◯ どちらでも大丈夫
```

---

## 設問5（任意）｜どちらの日も都合がつかない方へ

- **質問**：「どちらの日も都合がつかない」を選んだ方は、参加できそうな曜日・時間帯があれば教えてください
- **タイプ**：記述式（段落）
- **必須**：いいえ

```
設問3で「どちらの日も都合がつかない」を選んだ方へ。
もし他に参加できそうな曜日・時間帯があれば教えてください（できる範囲で調整します）
```

---

- **質問**：参加メンバー用のLINEグループに参加していますか？
- **タイプ**：ラジオボタン（単一選択）
- **必須**：はい

```
本番までの連絡は、まずLINEグループで行います。
参加メンバー用のLINEグループに参加していますか？

◯ すでに参加している
◯ まだ参加していない（このあと参加します）
```

> フォーム末尾、または「回答後の確認画面メッセージ」に LINEグループの参加リンクを記載してください。
> 例）まだの方はこちら → 【LINEグループ招待URL】
