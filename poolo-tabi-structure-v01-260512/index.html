この構造資料の目的
POOLO旅企画シリーズ（鞆の浦・東川・今後の拠点）に通底する不変軸を、ビジョン／価値観／ルールの3階層で整理する。上から下に行くほど具体になり、上位の言葉が下位の言葉を生む。
PART 1

ビジョン層

POOLO旅企画の不変軸。シリーズ全体の旗印として、LP・SNS・憲章で外に出す層。
VISION 01
旅を消費から文化に戻す
POOLOの社会運動。観光消費の文脈に巻き込まれないための立ち位置宣言。
VISION 02
受け取る人から、創る人へ
旅で起こる不可逆変化。受け取った人が、いつか誰かに何かを差し出す側に立つ。
PART 2

価値観層

POOLOバリュー（スタンス3）を旅文脈に翻訳した3つの軸。全員が共有する旅人の構え。
VALUE 01
自分で決めて、
自分で動く
POOLOバリュー「自己決定」に対応。ガイドや旅程に流されず、自分の判断で旅をつくる。
VALUE 02
訪れた地域を、
自分ごとにする
POOLOバリュー「自分ごと化」に対応。訪れた地域の文化・経済・環境を、自分のこととして引き受ける。
VALUE 03
「いま、ここ」を
面白がる
POOLOバリュー「面白がる／受容」に対応。計画外も、すれ違いも、土地の時間に乗って楽しむ。
PART 3

ルール層

具体的な振る舞い17個プール。価値観ごとに紐づける。
VALUE 01に紐づく 自分で決めて、自分で動く
  • 個人旅行をする（パッケージツアーに参加しない）
  • 余白のある旅をする、旅程を詰め込みすぎない
  • オフシーズンに旅をする
VALUE 02に紐づく 訪れた地域を、自分ごとにする
  • 地元の食材を食べる
  • 地域にお金を落とす（チェーン店を使わない）
  • 歴史を学んでから訪問する
  • 現地の人にリスペクトの気持ちを持つ
  • 伝統文化に触れる
  • give&give
  • いったときより綺麗にして帰る
  • アメニティを必要以上に使わない
  • マイボトルを持つ
  • マイ箸を持つ
  • 歩く（CO2を排出する乗り物に乗らない）
VALUE 03に紐づく 「いま、ここ」を面白がる
  • 現地の人と交流する
  • 長期滞在する
  • 暮らす
PART 4

運用ルール

17個のルールから、旅ごとに自分の3つを選ぶ。構造そのものが価値観の体現になる。
OPERATION
17個から自分の3つを選ぶ
拠点ごとに違う3つが選ばれてもいい。参加者ごとに違う3つが選ばれてもいい。選び方は自由。
1
17個のルールが、ひとつのプールとして示される
↓
2
参加者（または拠点）が、3つを選ぶ
↓
3
選ぶ行為そのものが、価値観1「自分で決めて、自分で動く」の体現になる
入れ子の構造美
ルールを「与える」のではなく、「選んでもらう」
POOLOの絶対原則「導きをしない」と整合。ルールという外形を保ちつつ、選択を参加者に返す。
