FYI｜最近の参考メモ

REFERENCE 01

文化人類学的視点

by 大川内直子

エンジニアリング（計画的に進める）と、ブリコルール（即興的に進んでいく）。

  • エンジニアリング＝失敗の回避
  • ブリコルール＝失敗から学ぶ
「わけのわからない他者」や「わけのわからない自分」と、向き合い続けるための知恵としての文化人類学。ネガティブ・ケーパビリティ。安易な答えに飛びつかず、現場の中に身を浸し続けること。
REFERENCE 02

自由より、自在に生きる

by 近内悠太さん＆内田樹
  • 導かれたものに上手くのっていくのが「自在」
  • 自在には、遊び的なニュアンスがある
  • 遊びには本来、目的はない
REFERENCE 03

本当の自分？自分探しの旅は、いつから？

参考メモ

1997年の中教審（中央教育審議会）。
「自分探し」という語の社会的な広がりは、ここを起点に語られることが多い。

REFERENCE 04

弱いつながり｜検索ワードを探す旅

by 東浩紀（幻冬舎文庫）

人は、自分の知っている言葉でしか検索できない。
だから、知らない場所へ身を置く必要がある。旅とは、自分の検索ワードを更新するための時間。

  • 強い絆だけにいると、視野はいつのまにか閉じていく
  • 偶然出会う「弱いつながり」が、新しい問いを連れてくる
  • 家族や仕事仲間（強い絆）の外側に、ときどき身を置くこと
POOLO LIFEで言えば、関心探究型の仲間50-70人という「弱いつながり」が、自分の検索ワードを書き換えていく場になっている。
REFERENCE 05

目標逆算型と関心探究型

POOLO LIFEの共通言語

POOLO LIFEは「関心探究型」のための場として設計している。社会の主流は「目標逆算型」。マーケや発信の言葉を選ぶときに、どちらの人に向けて書いているのかをチームで意識的に分けたい。

目標逆算型
  • 先に目標を置き、逆算して進む
  • 最短距離・再現性・明確な成果
  • 知っている言葉でしか検索できない
関心探究型
  • 関心を起点に、答えを急がず探究
  • 問いを仲間と深める／脱線が栄養
  • 想像していなかった景色に辿り着く
関心探究型を、50-70名 × 8か月で一つの場に集める。一人では辿り着けない景色まで、仲間の引力で運ばれていく。これがPOOLO LIFEの設計思想。
